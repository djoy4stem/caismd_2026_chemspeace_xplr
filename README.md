# caismd_2026_chemspeace_xlplr
Instructor-ready materials for a 60-minute hands-on workshop on open-source approaches to chemical space exploration. Includes slides and Jupyter notebooks to build and interpret molecular space maps, visualize properties and SAR, and practice reproducible, resource-conscious workflows suitable for modest hardware.
The session will be help during the Computational Approaches in Secondary Metabolites Discovery's Hybrid workshop held March 24-26, 2026.
